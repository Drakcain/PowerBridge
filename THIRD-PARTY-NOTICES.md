This file is an attribution and release-hygiene notice. It is not a substitute for each dependency's own license text or metadata.

## Android Platform and Tooling

PowerBridge Android is built with:

* Gradle
* Android Gradle Plugin
* Kotlin
* Android SDK

## Android Libraries

The Android app currently depends on libraries including:

* AndroidX Core KTX
* AndroidX AppCompat
* Material Components for Android
* AndroidX ConstraintLayout
* AndroidX Security Crypto
* OkHttp
* Kotlin Coroutines for Android
* ZXing Android Embedded

## Windows Companion

The Windows Companion uses:

* PowerShell 7+
* WPF / .NET on Windows for the GUI path
* locally bundled or referenced QR generation support under `windows-companion/lib/`
