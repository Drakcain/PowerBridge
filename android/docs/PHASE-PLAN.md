# Android Phase Plan

The root project phase plan is the source of truth:

```text
../../docs/PHASE-PLAN.md
```

Current Android truth:

* `Local Wi-Fi Wake` is live.
* `Home Relay Server` is live for advanced user-owned relay setups.
* Remote method planning is reduced to three families:
  * `Old Android Device`
  * `Alexa / Google Voice Devices`
  * `My Own Server`
* Unreliable extra relay families were removed from active scope.

Next Android runtime work should focus on the `Old Android Device` relay path first.
